# Java Web实现基本登录页面

## Using Java web technology to realize basic login

## 启动准备

项目启动的`Tomcat`服务器使用的是`Maven`插件，

在启动之前应先配置`Maven`启动项

可以参考下图

![Maven config 1](img/img1.png "Maven config 1")

![Maven config 2](img/img2.png "Maven config 2")

# 启动项目
启动命令
```shell
mvn clean tomcat7:run
```
